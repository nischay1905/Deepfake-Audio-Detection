# Deepfake Audio Detection 

### MARS Club IIT Roorkee – Open Projects 2026

## Project Overview

This project develops a Deep Learning based system to classify audio recordings as:

- Genuine (Human Speech)
- Deepfake (AI-Generated Speech)

The model uses Mel Spectrograms and EfficientNetB0 for classification.

## Performance Results

| Metric | Value |
|----------|----------|
| Accuracy | 95.45% |
| F1 Score | 95.50% |
| EER | 4.47% |
